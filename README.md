# MOS-READY 🎯

> **Microsoft Office Specialist Word Exam Prep — From Confusion to Clarity**

MOS-READY is a Gen-Z-focused, gamified study app that prepares students for the Microsoft Office Specialist (MOS) Word certification exam. It integrates an embedded Microsoft Word environment, interactive quizzes, XP-based progression, mock exams with hints, and a paid practice exam feature.

## Features

1. **Step-by-Step Procedural Guidance** — Structured learning modules covering all MOS Word exam objectives
2. **Embedded MS Word Integration** — Launch and dock Microsoft Word inside the app for hands-on practice
3. **Gamified Learning** — Match games, drag-and-drop, XP points, streaks, and level progression
4. **Section Quizzes** — Quiz after each learning section with instant feedback
5. **Custom Resources** — Upload your own study materials, get AI analysis, and auto-generate quizzes
6. **Mock Quiz Section** — 20-question real-life scenario mock exams with answer hints, non-repeating per session, with revision history
7. **Practice Exam (Paid)** — 50-minute, 40-question timed exams (2 attempts per payment, ₦250 via Paystack)
8. **Progress Dashboard** — Track XP, completed modules, quiz scores, and streaks

## Tech Stack

- **Frontend**: HTML5, CSS3 (vanilla), JavaScript (ES6+)
- **Backend**: Node.js + Express
- **Database**: SQLite (via better-sqlite3)
- **MS Word Integration**: COM automation via PowerShell/child_process
- **Build**: Vite for dev server

## Getting Started

```bash
npm install
npm run dev
```
