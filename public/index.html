🧠 The P-A-I-N-T & M-A-R-G-I-N Mnemonics

Format Painter (PAINT)

  • P - Pick the original text.
  • A - Access the Home tab.
  • I - Identify the paintbrush.
  • N - Now double-click it.
  • T - Target new text to paint!

Page Layouts (MARGIN)

  • M - Move to Layout tab.
  • A - Adjust Margins first.
  • R - Rotate (Orientation).
  • G - Gauge paper Size.
  • I - Insert Columns if needed.
  • N - Navigate page breaks.
